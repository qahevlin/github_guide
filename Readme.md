# ACE Team GitHub Guidebook

An interactive, beginner-friendly web guide for learning Git and GitHub — built for the ACE Team at QAHE Ltd.

---

## Overview

This project is a visual, step-by-step GitHub guide designed to walk new team members through the entire Git workflow, from signing up for GitHub to advanced skills and best practices. Each step is presented as a clickable node that displays detailed instructions and a video guide.

---

## Features

- **Interactive mind map** — click any node to reveal step-by-step instructions
- **Video guides** — each node links to a relevant tutorial video
- **Full user journey** covering:
  - Getting started & environment setup
  - Cloning or creating a repository
  - Basic Git operations (branch, commit, push, status)
  - Collaboration via pull requests and code reviews
  - Merge strategies and conflict resolution
  - Advanced skills (rebase, reset, bisect, issues)
  - Team best practices (protected branches, CODEOWNERS, workflow)
- **Learning resources section** with links to official docs, interactive tools, books, video tutorials, and community forums

## How to Use

1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Click any node in the mind map to view instructions and a video guide for that step
4. Press **Escape** or click any empty area to close the info panel

---

## User Journey

| Step | Topic |
|------|-------|
| 1 | Getting Started — Sign up for GitHub |
| 2 | Setup Environment — Install Git, text editor, configure Git |
| 3 | Choose Your Terminal — Git Bash or VS Code terminal |
| 4 | Choose Your Path — Clone existing repo or create new |
| 5 | Basic Git Operations — Branch, commit, push, status |
| 6 | Collaboration — Collaborators, pull requests, code review |
| 7 | Merge Strategy — Merge PRs, resolve conflicts |
| 8 | Advanced Skills — Issues, rebase, reset, bisect |
| 9 | Best Practices — Protected branches, CODEOWNERS, team workflow |
